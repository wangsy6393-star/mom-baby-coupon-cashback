BabyCash - 母婴用品返利平台

产品介绍

BabyCash是一个专注于母婴用品的优惠券和返利平台，支持淘宝、京东、抖音、拼多多四大电商平台，为用户提供一站式的省钱解决方案。

用户只需粘贴商品链接，即可一键查询优惠券和返利信息，让购物更加省钱省心。

目标用户

  • 有婴儿的家庭，特别是需要购买纸尿裤的父母
  • 孕妇及哺乳期妈妈
  • 关注母婴用品优惠信息的消费者
  • 注重性价比的理性购物者

核心优势

  • 专注母婴用品，品类精准
  • 四大平台全覆盖，一站式查询
  • 纸尿裤返利高达10%，行业领先
  • 操作简单，一键查询优惠
  • 提现门槛低，到账速度快

核心功能

商品链接解析

支持解析淘宝、京东、抖音、拼多多四大平台的商品链接，自动识别商品信息和平台类型。

支持的链接格式：

  • • 淘宝：https://item.taobao.com/item.htm?id=1234567890
  • • 京东：https://item.jd.com/100008368345.html
  • • 拼多多：https://mobile.yangkeduo.com/goods.html?goods_id=123456789
  • • 抖音：https://haohuo.jinritemai.com/views/product/item2?id=345678901234

优惠券查询

部署说明

快速开始

1. 下载项目文件

您已经在以下路径拥有完整的项目文件：

/home/user/vibecoding/workspace/wechat-baby-rebate-bot-web

2. 查看项目结构

wechat-baby-rebate-bot-web/
├── app.py                 # Flask后端主程序
├── requirements.txt       # Python依赖包列表
├── README.md              # 项目说明文档
├── start.sh               # 启动脚本
├── templates/             # HTML模板目录
│   ├── index.html         # 首页
│   ├── dashboard.html     # 用户仪表盘
│   └── login.html         # 登录页面
└── demo.html              # 演示页面（当前页面）

3. 启动方式

由于当前环境网络限制，无法直接安装依赖包。在实际环境中，您可以通过以下方式启动： 

# 方法1：使用启动脚本
chmod +x start.sh
./start.sh

# 方法2：手动启动
pip install -r requirements.txt
python app.py

4. 访问方式

启动成功后，可以通过以下方式访问： 

# 首页
http://localhost:5000

# 用户仪表盘
http://localhost:5000/dashboard

# 登录页面
http://localhost:5000/login

5. 实际部署建议

在实际部署时，您需要：
1. 注册各电商平台联盟账号获取API权限
2. 创建MongoDB数据库存储用户数据
3. 配置微信公众号或其他登录方式
4. 部署到云服务器或使用容器化部署
