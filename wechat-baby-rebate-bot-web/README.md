# BabyCash - 母婴用品返利平台网页版

BabyCash是一个专注于母婴用品的优惠券和返利平台，支持淘宝、京东、抖音、拼多多四大电商平台，为用户提供一站式的省钱解决方案。

## 功能特性

### 核心功能
1. **商品链接解析**：支持解析四大平台的商品链接，自动识别商品信息
2. **优惠券查询**：自动查询商品可用的优惠券信息
3. **返利计算**：根据不同平台和商品类别自动计算返利金额
4. **用户余额管理**：实时显示用户返利余额和待返利金额
5. **提现功能**：支持余额提现，最低提现金额10元，提现手续费10%
6. **订单管理**：查看历史订单和返利记录
7. **热门商品推荐**：展示热门母婴商品的优惠信息

### 技术特点
1. **响应式设计**：支持PC端和移动端访问
2. **现代化UI**：采用Tailwind CSS构建现代化用户界面
3. **实时交互**：使用JavaScript实现流畅的用户交互体验
4. **模拟数据**：提供完整的模拟数据，可直接体验所有功能

## 安装部署

### 环境要求
- Python 3.6+
- pip包管理器
- MongoDB（可选，当前使用模拟数据）

### 安装步骤

1. **克隆项目**
```bash
git clone https://your-repository-url/wechat-baby-rebate-bot-web.git
cd wechat-baby-rebate-bot-web
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **运行项目**
```bash
python app.py
```

4. **访问项目**
打开浏览器访问 `http://localhost:5000`

### 环境变量配置

项目支持通过环境变量进行配置，可创建 `.env` 文件：

```
# MongoDB配置（可选）
MONGO_URI=mongodb://localhost:27017/baby_rebate_bot

# Flask配置
FLASK_ENV=development
FLASK_SECRET_KEY=your-secret-key
```

## 返利规则

### 支持平台及返利比例
- **淘宝/天猫**：纸尿裤8%，孕妈用品6%，婴儿用品7%，其他5%
- **拼多多**：纸尿裤9%，孕妈用品7%，婴儿用品8%，其他6%
- **抖音**：纸尿裤10%，孕妈用品8%，婴儿用品9%，其他7%
- **京东**：纸尿裤7%，孕妈用品5%，婴儿用品6%，其他4%

### 支持品牌
- 纸尿裤：Babycare、好奇、大王、新贝、尤妮佳、露安适、帮宝适、花王、moony等
- 孕妈用品：好孩子、贝亲、可优比等知名品牌
- 婴儿用品：各类知名婴儿奶粉、辅食、洗护用品等

### 提现规则
- 最低提现金额：10元
- 提现手续费：10%
- 提现方式：微信零钱、银行卡
- 到账时间：微信零钱实时到账，银行卡1-3个工作日

## 常见问题

1. **为什么有些商品没有优惠信息？**
   - 可能是因为该商品不在支持的品牌范围内
   - 或者当前没有可用的优惠券
   - 只支持指定品牌的母婴用品

2. **返利什么时候到账？**
   - 确认收货后，系统会自动将返利发放到账户余额
   - 一般在确认收货后24小时内完成

3. **如何提现？**
   - 当账户余额达到10元以上时，可以申请提现
   - 支持微信零钱提现，提现手续费为10%

4. **支持哪些电商平台？**
   - 目前支持淘宝/天猫、京东、抖音、拼多多四大电商平台
